# nCine Presentation Template

A Slidev presentation documenting a 14-year journey developing nCine, an open-source 2D game framework. Presented at /dev/games/2025.

## Tech Stack

- **Framework**: Slidev v52
- **Language**: Vue 3 / Markdown
- **Package Manager**: npm
- **Build Output**: `dist`

## Available Skills

| Skill | Description |
|-------|-------------|
| `build-and-deploy` | Build and deploy the presentation to Vercel or Netlify |

## Quick Start

```bash
npm install
npm run dev
```

## Customization

- Edit `slides.md` to modify the presentation content
- Slidev uses Markdown with Vue components for slides
- See [Slidev documentation](https://sli.dev) for more features
