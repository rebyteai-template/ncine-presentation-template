# AI Agent Instructions for NCine Presentation Template

This file provides guidance for AI coding agents customizing this Vue.js presentation template.

## Quick Start

1. **Slides**: Edit Vue components for slide content
2. **Styling**: Customize CSS/styles
3. **Assets**: Add images to asset folders

### Assets

## Build and Test

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

---

## Deployment Notes

**Build output**: `dist/` folder

Deploy to any static hosting service.
